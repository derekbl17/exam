# EXAM help desk app

Exams help desk app

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file in root folder

`NODE_ENV = development`

`PORT`

`MONGO_URI`

`JWT_SECRET`

## Build and run

To build, from root:

```bash
  npm run build
```

Then launch:

```bash
  npm run dev
```
